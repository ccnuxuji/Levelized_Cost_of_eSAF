# eSAF技术经济分析模型 - Streamlit网页应用

这是一个基于Streamlit的网页应用，用于分析电子合成可持续航空燃料（eSAF）的技术经济性能。

## 🚀 快速开始

### 1. 安装依赖包

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run streamlit_app.py
```

### 3. 在浏览器中访问

应用将在本地启动，通常地址为：`http://localhost:8501`

## 📊 功能特点

### 核心分析功能
- **平准化成本计算**: 完整的eSAF成本建模
- **成本结构分析**: CAPEX vs OPEX，各阶段成本分解
- **敏感性分析**: 电力价格和生产规模敏感性
- **盈亏平衡分析**: 与传统航空燃料的比较
- **可视化图表**: 交互式图表和数据展示

### 技术路径
- **固定配置**: Fischer-Tropsch合成路径
- **CO2来源**: 直接空气捕获 (DAC)
- **完整链条**: DAC → 电解 → FT合成 → 分销
- **功能单位**: USD/MJ

## 🔧 使用方法

### 步骤1: 设置参数
在左侧边栏中调整以下参数组：
- **基本经济参数**: 折现率、项目寿命、产能利用率等
- **DAC成本参数**: 设备成本、电力成本、热能成本等
- **电解成本参数**: CO2电解和水电解设备成本
- **FT合成参数**: 反应器成本、催化剂成本等
- **分销参数**: 运输距离和成本

### 步骤2: 运行分析
点击"🚀 运行TEA分析"按钮开始计算

### 步骤3: 查看结果
在不同标签页中查看：
- **成本分析**: 成本分解和结构分析
- **敏感性分析**: 关键参数影响分析
- **盈亏平衡**: 与传统燃料比较
- **详细结果**: 完整数据和下载功能

## 📈 分析结果

### 关键输出指标
- 平准化成本 (USD/MJ)
- 燃料成本 (USD/L)
- 各阶段成本分解
- 成本贡献百分比
- 盈亏平衡碳税

### 敏感性分析
- **电力价格影响**: 分析不同电力价格下的成本变化
- **生产规模效应**: 探索规模经济效应
- **参数敏感性**: 识别关键成本驱动因素

## 🏗️ 模型架构

```
DAC (直接空气捕获)
    ↓ CO2
电解系统 (CO2电解 + 水电解)
    ↓ 合成气 (CO + H2)
Fischer-Tropsch合成
    ↓ 液体燃料
分销系统
    ↓ 最终产品
```

## 🔬 技术说明

### 模型假设
- Fischer-Tropsch合成效率: 基于文献数据
- 能量转换效率: 考虑实际工业应用
- 设备寿命: 20年标准工业寿命
- 产能利用率: 90%（可调节）

### 成本计算方法
- **CAPEX年化**: 使用资本回收因子(CRF)
- **OPEX计算**: 固定成本 + 变动成本
- **平准化成本**: 总年成本 / 年产能（MJ）
